1. W jaki sposób można określić właściwość danego obiektu (dwa sposoby wraz z przykładami)?

Sposób 1: Korzystanie z notacji kropkowej.


auto.marka = "Opel Omega";
auto.rok = 1996;

Sposób 2: Korzystanie z notacji nawiasów kwadratowych.


auto["marka"] = "Opel Omega";
auto["rok"] = 1996;

2. W jaki sposób można odwołać się do pól (właściwości) i metod?

Odwołujemy się do pól i metod, poprzedzając ich nazwę nazwą obiektu oraz kropką.


console.log(samochod1.kolor); // Wyświetla kolor obiektu samochod1

3. Wymień cechy charakterystyczne konstruktora.

  • Wywoływany automatycznie podczas tworzenia obiektu.
  • Nazywa się tak samo jak klasa.
  • Nie zwraca żadnej wartości.
  • Może przyjmować argumenty, aby inicjalizować obiekt.

4. Konstruowanie obiektu na podstawie przykładu

Obiekt z trzema polami i jedną metodą:


var osoba_1 = {
    nazwisko: "Kowalski",
    imie: "Jan",
    zawod: "piekarz",
    wyswietl: function () {
        document.write(this.nazwisko + " " + this.imie);
    }
};

5. Znaczenie słowa this. Jak oddzielamy pola oraz metody?

Słowo kluczowe this odnosi się do bieżącego obiektu, umożliwiając dostęp do jego pól i metod.

Pola i metody w obiekcie oddzielamy przecinkami.


var osoba = {
    nazwisko: "Nowak",
    wyswietl: function () {
        console.log(this.nazwisko);
    }
};

6. Konstruktor w JS na podstawie przykładu wraz z opisem.

Przykład:


function Klient(nazwisko, imie, zawod) {
    this.nazwisko = nazwisko;
    this.imie = imie;
    this.zawod = zawod;
    this.wypisz = function () {
        alert(this.nazwisko + " " + this.imie);
    };
}

Opis: Konstruktor definiuje szkielet obiektu, przypisując parametry do pól za pomocą this. Służy do tworzenia nowych obiektów.

7. Znaczenie słowa kluczowego new wraz z przykładem.

new tworzy nowy obiekt na podstawie konstruktora, przypisując wartości do jego pól.


var osoba1 = new Klient("Kowalski", "Jan", "kierowca");

8. Znaczenie słowa kluczowego prototype wraz z przykładem i opisem.

Właściwość prototype pozwala dodawać nowe metody lub właściwości do istniejących obiektów.


function Klient() {
    this.nazwisko = "Bielski";
    this.imie = "Paweł";
}
Klient.prototype.pisz_dane = function () {
    document.write(this.nazwisko + " " + this.imie);
};
Klient.prototype.zawod = "kierowca";
var osoba1 = new Klient();
osoba1.pisz_dane(); // Wyświetli: Bielski Paweł
